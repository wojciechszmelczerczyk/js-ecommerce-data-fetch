## To run

Clone repository.

```sh
git clone https://github.com/wojciechszmelczerczyk/js-ecommerce-data-fetch.git
```

Navigate to project root directory.

```sh
cd ./js-ecommerce-data-fetch
```

Install dependencies.

```sh
npm i
```

Run server.

```sh
npm run start
```
